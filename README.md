## Quick Start

1. **Installation**
   ```bash
   npm install
   ```

2. **Configuration**
   ```bash
   cp .env.example .env
   # Edit .env with your settings
   ```

3. **Start the Bot**
   ```bash
   npm start
   ```

## Development

1. **Test Environment**
   ```bash
   npm run dev
   ```

2. **Run Tests**
   ```bash
   npm test
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
